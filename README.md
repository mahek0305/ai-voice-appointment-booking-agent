# 🤖 AI Voice Appointment Booking Agent


An AI-powered voice appointment booking system built using **n8n** and **Vapi** 

## 📌 Overview

This project automates appointment scheduling through a voice-based AI assistant. The workflow checks appointment availability, books appointments, and sends confirmation emails using n8n automation.

## ✨ Features

- 🎙️ AI Voice Assistant using Vapi
- ⚙️ Workflow Automation with n8n
- 📅 Appointment Availability Check
- 📝 Automated Appointment Booking
- 📩 Email Notifications
- 🧩 AI Function Calling

## 🛠️ Tech Stack

- n8n
- Vapi
- OpenAI
- Gmail
- Webhooks
